Инструкция по использованию калькулятора

Данный калькулятор предназначен для вычисления операций вычитания и умножения больших чисел.

Ограничения по вводу:

  • Длина числа не более 64 символов
  • Число может содержать только знаки "+" и "-"(первым символом) и цифры

Для получения результата:

  1. Введите первое число в поле "Первое число"
  2. Введите второе число в поле "Второе число"
  3. Выберите действие в выпадающем поле "Операция": "-" - вычитание, "*" - умножение
  4. Нажмите кнопку "ОК" для того, чтобы получить результат
  5. Результат появится в строке "Результат"

Примечание. Если какое-то из полей подсвечено красным, значит, в нем есть ошибка. Проверьте его на соответствие ограничениям, указанным выше.
